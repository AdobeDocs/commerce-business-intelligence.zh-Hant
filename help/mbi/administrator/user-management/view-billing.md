# 帳單資訊

若要檢視、新增或變更帳戶的帳單資訊，您必須擁有 `Billing` 許可權。 [許可權是逐位使用者設定的](../../administrator/user-management/user-management.md) 建立使用者帳戶時。 如果您需要 `Billing` 許可權，請聯絡您帳戶的管理員。
