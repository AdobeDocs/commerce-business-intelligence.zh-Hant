# 帳單資訊

若要檢視、新增或變更帳戶的帳單資訊，您必須擁有`Billing`許可權。 [建立使用者帳戶時，會逐個使用者設定許可權](../../administrator/user-management/user-management.md)。 如果您需要`Billing`許可權，請連絡您帳戶的管理員。
