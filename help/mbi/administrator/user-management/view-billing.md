# 帳單資訊

若要檢視、新增或變更帳戶的帳單資訊，您必須 `Billing` 權限。 [權限是依使用者而設定](../../administrator/user-management/user-management.md) 建立使用者帳戶時。 如果您需要 `Billing` 權限，請連絡您帳戶的管理員。
