# 計費資訊

要查看、添加或更改帳戶的帳單資訊，您必須 `Billing` 權限。 [權限是按用戶設定的](../../administrator/user-management/user-management.md) 建立用戶帳戶時。 如果你需要 `Billing` 權限，請與您帳戶的管理員聯繫。
