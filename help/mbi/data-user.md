# 資料使用者

您可使用資料來協助進行業務決策。 您可能會收到團隊中資料分析師的報表和控制面板，但您也想要了解如何建立這些報表和控制面板。

在本節中，您將學習如何在 [!DNL MBI] 使用報表和控制面板，以及如何透過電子郵件和 `.csv` 檔案。
