# 預測

能夠預測客戶在下更多訂單時的生命週期價值是任何規模業務的最重要方面之一。

在本節中，您將瞭解當前客戶的生存期價值，並預測隨著訂單的增加，生存期價值將如何增長。

另請參閱： [預期生存期值(LTV)分析（基本）](../../data-analyst/analysis/ess-expected-ltv.md)
