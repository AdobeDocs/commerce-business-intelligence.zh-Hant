# 預測

能夠預測客戶下多筆訂單時的存留期價值，是任何規模企業最重要的方面之一。

在本節中，您將瞭解目前客戶的期限值，並預測期限值如何隨著更多訂單而增加。

另請參閱： [預期期限值(LTV)分析（基本）](../../data-analyst/analysis/ess-expected-ltv.md)
