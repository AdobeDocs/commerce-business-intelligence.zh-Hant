# 使用整合連線外部資料

整合是資料來源。 整合是可連線至資料倉庫的應用程式和資料庫。

在本節中，我們會提供可用的整合功能，讓您將各種外部資料來源連結至 [!DNL MBI].

若要刪除連線或整合，請聯絡 [支援](../../../guide-overview.md) 並加入要刪除的連線或整合名稱。
