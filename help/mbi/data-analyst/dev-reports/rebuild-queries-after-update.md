# 更新週期後重建查詢

您不必重新生成查詢。 使用 [`SQL Report Builder`](../dev-reports/sql-rpt-bldr.md) 被保存得跟傳統 `Report Builder`。 SQL圖表的更新過程相同 — 更新資料後，將重新計算並重新顯示圖表中的值。
