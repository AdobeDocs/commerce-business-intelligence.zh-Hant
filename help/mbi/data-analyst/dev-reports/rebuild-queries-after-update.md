# 在更新週期後重建查詢

您不需要重新建立查詢。 使用建立的報表 [`SQL Report Builder`](../dev-reports/sql-rpt-bldr.md) 會像在傳統檔案中建立的檔案一樣儲存 `Report Builder`. SQL圖表的更新程式相同 — 更新資料後，圖表中的值將會重新計算並重新顯示。
