# 在更新週期後重建查詢

您不必重建查詢。 使用 [`SQL Report Builder`](../dev-reports/sql-rpt-bldr.md) 會像傳統 `Report Builder`. SQL圖表的更新過程完全相同 — 更新資料後，將重新計算並重新顯示圖表中的值。
