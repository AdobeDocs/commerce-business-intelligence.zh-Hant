# 存取共用控制面板

讓新用戶快速上手非常簡單。

## 新用戶

新使用者建立密碼後，熱門共用控制面板清單隨即顯示。 他們可以選取要在控制面板上顯示的控制面板，然後按一下 **[!UICONTROL Done]**.

## 現有使用者

若要再次檢視共用控制面板的清單，使用者可以按一下 **[!UICONTROL Find]** 在 `Global Search` 功能表。
