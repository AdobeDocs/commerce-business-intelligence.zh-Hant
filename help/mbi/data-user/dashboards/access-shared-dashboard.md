# 存取共用儀表板

讓新使用者快速上手很簡單。

## 新使用者

新使用者建立密碼後，熱門的共用儀表板清單隨即顯示。 他們可以選取想要在儀表板上顯示的儀表板，然後按一下 **[!UICONTROL Done]**.

## 現有使用者

若要再次檢視共用儀表板的清單，使用者可以按一下 **[!UICONTROL Find]** 在 `Global Search` 功能表。
