# 變更共用控制面板的存取權

您可以隨時使用相同的步驟變更控制面板的存取權， [最初共用控制面板](../../data-user/dashboards/share-dashboard-with-users.md){:target=&quot;_blank&quot;}。 在 `Share Dashboard` 功能表，視需要更新使用者/組織權限，然後按一下 **[!UICONTROL Save Changes]** 以更新。
