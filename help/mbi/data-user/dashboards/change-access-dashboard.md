# 變更共用儀表板的存取權

您可以隨時變更儀表板的存取權，使用與以下專案相同的步驟： [最初共用控制面板](../../data-user/dashboards/share-dashboard-with-users.md){： target=&quot;_blank&quot;}。 顯示在中的使用者清單中 `Share Dashboard` 功能表，視需要更新使用者/組織許可權，然後按一下 **[!UICONTROL Save Changes]** 以更新。
