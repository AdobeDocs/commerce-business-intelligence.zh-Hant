# 更改對共用儀表板的訪問

可以隨時更改對儀表板的訪問，使用相同的步驟 [最初共用儀表板](../../data-user/dashboards/share-dashboard-with-users.md){:target=&quot;_blank&quot;}。 在中顯示的用戶清單中 `Share Dashboard` 菜單，根據需要更新用戶/組織權限，然後按一下 **[!UICONTROL Save Changes]** 的子菜單。
