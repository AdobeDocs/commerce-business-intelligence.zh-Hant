# 變更共用儀表板的存取權

您可以隨時使用相同的步驟變更儀表板的存取權，以[開始共用儀表板](../../data-user/dashboards/share-dashboard-with-users.md){： target=&quot;_blank&quot;}。 在`Share Dashboard`功能表中顯示的使用者清單中，視需要更新使用者/組織許可權，然後按一下&#x200B;**[!UICONTROL Save Changes]**&#x200B;以更新。
