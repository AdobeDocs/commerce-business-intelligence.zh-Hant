# 保留（取消共用）控制面板

更換團隊？ 在春天的中間 [!DNL MBI] 帳戶？ 若要離開控制面板或取消共用您自己，請按一下 **[!UICONTROL Shared]** 在畫面頂端，當您要離開控制面板時。 按一下 **[!UICONTROL Leave this Dashboard]** 把自己移走。
