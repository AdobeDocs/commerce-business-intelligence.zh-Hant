# 保留（取消共用）儀表板

更換團隊？ 在春天的中間 [!DNL Commerce Intelligence] 賬戶？ 要離開或取消共用儀表板，請按一下 **[!UICONTROL Shared]** 在螢幕頂部，在儀表板中要離開。 按一下 **[!UICONTROL Leave this Dashboard]** 去除你自己。
