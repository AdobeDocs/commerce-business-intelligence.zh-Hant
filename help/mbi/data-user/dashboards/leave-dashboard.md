# 保留（取消共用）控制面板

變更團隊？ 在春天中清洗您的 [!DNL Commerce Intelligence] 帳戶？ 若要離開儀表板或取消共用您自己，請按一下 **[!UICONTROL Shared]** 位於熒幕頂端時，如果您想離開儀表板。 按一下 **[!UICONTROL Leave this Dashboard]** 以移除您自己。
