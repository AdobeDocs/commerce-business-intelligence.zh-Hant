# 設定預設儀表板

預設操控板是開啟時看到的第一個操控板 [!DNL Commerce Intelligence]。

1. 在操控板中，按一下 **[!UICONTROL Dashboard Options]** 位於螢幕頂部。

1. 按一下 **[!UICONTROL Make Default]** 的下界。

1. 顯示確認提示時，按一下 **[!UICONTROL Yes]** 的子菜單。

這現在是您的新預設儀表板。
