# 建立報表副本

通常，您會發現自己想要建立類似定義的圖表，其中您只想更改一個元素，例如編輯篩選器或更改 `group by`。

在這些情況下，啟動 `Chart Editor` 按一下 **[!UICONTROL Save As]** 在右上角。 這將複製現有圖表，並使用您選擇的新名稱將其添加到當前儀表板中，並允許您立即編輯新圖表的設定。
