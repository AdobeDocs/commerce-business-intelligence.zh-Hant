# 建立報表副本

通常，您會發現自己想要建立類似定義的圖表，而您只想變更一個元素，例如編輯篩選器或變更 `group by`.

在這些情況下，請啟動 `Chart Editor` 按一下 **[!UICONTROL Save As]** 在右上角。 這會複製您現有的圖表，並以您選取的新名稱將其新增至目前的控制面板，並讓您立即編輯新圖表的設定。
