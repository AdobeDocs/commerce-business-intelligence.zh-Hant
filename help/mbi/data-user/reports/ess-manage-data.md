# 管理資料

「管理資料」提供了對各種工具的訪問權限，這些工具用於管理整合、報告和圖表資料、儀表板和導出。

## 訪問 [!DNL Manage Data]:

1. 在菜單中，按一下 **[!DNL Manage Data]**。

1. 在提要欄中，從以下標題中選擇您想要的主題：

   * `Data Sources`
   * `Manage Data`
   * `Export Data`
   ![管理資料](../../assets/magento-bi-manage-data.png)<!--{: .zoom}-->
