# 管理資料

「管理資料」可讓您存取各種工具，以管理整合、報表和圖表資料、控制面板及匯出。

## 若要存取 `Manage Data`:

1. 在功能表中，按一下 **[!DNL Manage Data]**.

1. 在側欄的下列標題下，選擇您要的主題：

   * `Data Sources`
   * `Manage Data`
   * `Export Data`
