# 匯出圖表資料

如果您想要檢檢視表後的資料，請按一下任何圖表右上角的齒輪圖示。

選取：

- `Report Export`
- `Full .csv Export`
- `Full Excel Export`
- `Raw Export`

您也可以定義 [要包含在匯出中的表格和欄](../../tutorials/export-raw-data.md).

如果您需要在簡報或檔案中包含圖表，可以將圖表匯出為PNG、PDF或SVG格式的影像。 按一下任何圖表右上角的齒輪圖示。 選取 `Enlarge` 選項，然後按一下 **[!UICONTROL Download]** 以完成匯出。
