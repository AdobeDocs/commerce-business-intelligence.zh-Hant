# 導出圖表資料

如果要查看圖表後面的資料，請按一下任意圖表右上角的齒輪表徵圖。

選擇以下任一項：

- `Report Export`
- `Full .csv Export`
- `Full Excel Export`
- `Raw Export`

也可以定義 [要包括在導出中的表和列](../../tutorials/export-raw-data.md)。

如果需要在演示文稿或文檔中包含圖表，可以將圖表導出為PNG、PDF或SVG格式的影像。 按一下任意圖表右上角的齒輪表徵圖。 選擇 `Enlarge` 選項，然後按一下 **[!UICONTROL Download]** 完成導出。
