# 導出圖表資料

如果要查看圖表後的資料，請按一下任何圖表右上角的齒輪表徵圖。

選擇以下任一項：

- `Report Export`
- `Full .csv Export`
- `Full Excel Export`
- `Raw Export`

您也可以定義 [要包括在導出中的表和列](../../tutorials/export-raw-data.md).

如果需要在演示文稿或文檔中包含圖表，可以按一下圖表右上角的齒輪表徵圖，將圖表導出為PNG、PDF或SVG格式的影像。 選取 `Enlarge` 選項，然後按一下 **[!UICONTROL Download]** 來完成匯出。
